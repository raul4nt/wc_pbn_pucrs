## Sobre o Projeto
Este projeto é uma implementação simplificada do utilitário `wc` (word count) nativo de sistemas Unix/Linux, desenvolvido na linguagem C. O programa lê um arquivo de texto informado pelo usuário e realiza a contagem de quebras de linha, palavras e caracteres totais.

## Compilação
Para compilar o código-fonte, utilize o `gcc`:

```bash
gcc wc.c -o wc
```

## Execução
Passe o arquivo de texto como argumento na execução:

```bash
./wc <nome_do_arquivo.txt>
```
